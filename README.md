# 介绍
这是我学习完VUE全家桶之后练习的项目，豆瓣部分UI是自己写的，感觉好累。。后面的网易云部分准备用vant来写UI，不自己写了。项目中的图片加载使用了vue的插件vue-lazyload，下拉加载更多图片是自己写的。豆瓣的搜索功能是没有做的，因为豆瓣的搜索API最近关闭了。

# 用到的知识
vue + axios + vue-router + Vuex + vue-cli
